# Memories and Miles

A lightweight static HTML/CSS/JavaScript site for planning Disney, Universal, and cruise vacations. Features a tropical vacation aesthetic designed to inspire users to book their dream vacation.

## Features

- **Home page** with gradient hero section and feature cards
- **Interactive questionnaire** with conditional logic to help choose a destination
- **Contact form** that opens an email to laura@whitneyworldtravel.com
- **Tropical vacation design** with vibrant colors and smooth animations
- **Fully responsive** with mobile-first design
- **No build process** - pure static files

## Local Development

Run with any static file server:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000

# Or use VS Code Live Server extension
```

Then visit `http://localhost:8000`

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
